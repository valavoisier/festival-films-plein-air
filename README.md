#  Gestion de projet et conception du site web du festival “Les Films de Plein Air”

## Analyse des besoins & maquette HTML/CSS
Analyse des besoins du client pour son festival de films d'auteur en plein air du 5 au 8 août au parc Monceau à Paris, de 18 heures à minuit. Le budget limité. L'association a besoin de communiquer en ligne sur son festival, d'annoncer les films projetés et de recueillir les réservations (accès gratuit).
Voici ce qu'elle vous dit :
Mon association "Les Films de Plein Air" vient d'obtenir l'autorisation de projeter au parc Monceau cette année, chaque soir du 5 au 8 août, . Je souhaite ainsi faire découvrir des films d'auteur au grand public.
## Fonctionnalités prévues
- Présentation du festival
- Programme des projections (fiches films)
- Actualités
- Formulaire de pré‑inscription
- Mentions légales
- Navigation interne multi‑pages
- Responsive design

## Fichiers

- **index.html** : La page d'accueil du site web, contenant toutes les information sur le festival.
- **styles/style.css** : Fichier CSS pour définir l'apparence générale du site header footer et menus.
- **styles/index.css** : Fichier CSS pour définir l'apparence des sections de la page d'accueil.
- **styles/programme.css** : Fichier CSS pour définir l'apparence de la page programme et de la page actualités.
- **styles/single.css** : Fichier CSS pour définir l'apparence des pages de chaque film.
- **styles/contact.css** : Fichier CSS pour définir l'apparence de la page contact.
- **styles/mentions-legales.css** : Fichier CSS pour définir l'apparence de la page des mentions légales.
- **livrables/cahier-des-charges.pdf

## Instructions

1. Clonez le dépôt ou téléchargez les fichiers.
2. Ouvrez `index.html` dans un navigateur pour visualiser le site.
3. Modifiez `styles/style.css` et `styles/index.css` pour personnaliser l'apparence du site.
